Career Navigator

Career Navigator is a simple and interactive Python-based project that helps users discover potential career paths based on their preferences, interests, strengths, and working style. It provides personalized career suggestions and a visual progress bar that shows your interest level across different fields.


About the Project

This project asks a series of questions such as:

What kind of workspace do you prefer?

How do you solve problems?

What subjects do you enjoy the most?

What type of tasks make you feel confident?

Do you like creative, technical, or management-oriented work?


Based on your answers, the program analyses your responses and suggests possible careers that match your personality and interests. It also displays your interest distribution using a simple progress bar-like visual.

How It Works

1. The program asks the user a set of questions.


2. Each answer earns points for certain career categories.


3. Points are compared to determine the best career suggestions.


4. A simple progress bar is shown to display interest distribution.
